# AI Chat Assistant

## Service Name
NTA Academy AI Assistant (Chatbot)

## Output Sent from System (Input to LLM)
- **Context**: User role (Trainee, Admin, or Content Editor).
- **System Prompt**: A specialized persona definition (e.g., "You are the NTA Academy Assistant for Trainees...").
- **User Input**: The specific question or message typed by the user.
- **Conversation History**: Previous messages in the session for context-aware responses.

## Input Taken from LLM (Output from LLM)
- **Response**: A text-based answer, formatted in Markdown, providing guidance, answering system queries, or explaining recruitment stages.

## Currently Displayed
- **Trainee Portal**: Floating chat widget on the bottom right of the dashboard.
- **Admin Portal**: Sidebar assistant for managing candidates and understanding system analytics.
- **Editor Portal**: Contextual help for content management.
