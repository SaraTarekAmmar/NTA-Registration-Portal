Session 1: Python Review for Data Science

Welcome to the first session! Before we dive into heavy data libraries like Pandas and NumPy, we must ensure our "Core Python" skills are sharp. Data science often requires writing efficient, readable, and functional code to transform raw data.

1. List Comprehensions

List comprehensions provide a concise way to create lists. In data science, we use them constantly for cleaning strings or transforming data columns.

Syntax: [expression for item in iterable if condition]

Example:

# Traditional way
squares = []
for x in range(10):
    squares.append(x**2)

# List comprehension way
squares = [x**2 for x in range(10)]

# With a condition (Only even squares)
even_squares = [x**2 for x in range(10) if x % 2 == 0]


2. Lambda Functions

Lambda functions are small, anonymous functions defined with the lambda keyword. They are most useful when passed as arguments to functions like map(), filter(), or Pandas .apply().

Example:

# A function that adds 10 to a number
add_ten = lambda x: x + 10
print(add_ten(5)) # Output: 15

# Used in sorting (sorting a list of tuples by the second element)
data = [("Apple", 15), ("Banana", 7), ("Cherry", 20)]
data.sort(key=lambda x: x[1])
print(data) # Output: [('Banana', 7), ('Apple', 15), ('Cherry', 20)]


3. Decorators (Brief Overview)

Decorators are a way to modify the behavior of a function without changing its source code. In data science, they are often used for logging, timing execution, or authentication.

Example: A Simple Timer Decorator

import time

def timer_decorator(func):
    def wrapper(*args, **kwargs):
        start_time = time.time()
        result = func(*args, **kwargs)
        end_time = time.time()
        print(f"Execution time of {func.__name__}: {end_time - start_time:.4f} seconds")
        return result
    return wrapper

@timer_decorator
def heavy_computation():
    return sum(range(10**7))

heavy_computation()


4. Map, Filter, and Reduce

These functional programming tools are essential for data transformation.

map(func, seq): Applies a function to all items in an input list.

filter(func, seq): Filters elements of a sequence based on a boolean condition.

Example:

numbers = [1, 2, 3, 4, 5, 6]

# Convert all numbers to strings
str_nums = list(map(str, numbers))

# Keep only numbers greater than 3
big_nums = list(filter(lambda x: x > 3, numbers))
