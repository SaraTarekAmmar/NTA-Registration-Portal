Session 5: Handling Missing Data and Data Cleaning Techniques

Welcome to Week 3! In the previous sessions, we worked with "perfect" datasets. In the real world, data is messy: it has missing values, duplicates, and incorrect types. Today, we learn the essential art of Data Cleaning.

1. Detecting Missing Data

Pandas primarily uses NaN (Not a Number) to represent missing data. We use isnull() or isna() to find them.

import pandas as pd
import numpy as np

# Create a messy DataFrame
data = {
    'Name': ['Alice', 'Bob', 'Charlie', 'Dave', None],
    'Age': [25, np.nan, 35, 40, 28],
    'Score': [85, 90, np.nan, 70, 88]
}
df = pd.DataFrame(data)

# Check for nulls in each column
print(df.isnull().sum())

# View rows with any missing values
print(df[df.isnull().any(axis=1)])


2. Dealing with Missing Values

You generally have three choices when encountering missing data:

Option A: Drop them

Use dropna() if the missing data is sparse and won't significantly bias your results.

# Drop any row containing a NaN
df_cleaned = df.dropna()

# Drop rows only if 'Score' is missing
df_subset = df.dropna(subset=['Score'])


Option B: Fill them (Imputation)

Use fillna() to replace NaN with a specific value, like the mean, median, or a placeholder.

# Fill missing ages with the average age
mean_age = df['Age'].mean()
df['Age'] = df['Age'].fillna(mean_age)

# Fill categorical missing values with 'Unknown'
df['Name'] = df['Name'].fillna('Unknown User')


3. Handling Duplicates

Duplicate records can skew your analysis.

# Find duplicate rows
print(df.duplicated())

# Remove duplicates
df = df.drop_duplicates()
