Session 6: GroupBy Operations and Pivot Tables

Welcome to the final session of Week 3! Now that we know how to clean our data, it's time to learn how to summarize it. In Data Science, we often need to look at data in groups—for example, calculating the average sales per region or the total number of users per subscription tier.

Today, we cover the "Split-Apply-Combine" strategy using GroupBy and the powerful Pivot Table.

1. The Split-Apply-Combine Strategy

When we use groupby(), Pandas performs three steps:

Split: Breaks the data into groups based on a key (e.g., 'Category').

Apply: Computes a function (like mean, sum, or count) for each group.

Combine: Merges the results into a new summary object.

import pandas as pd

# Sample Data
data = {
    'Department': ['Sales', 'IT', 'Sales', 'IT', 'HR', 'HR'],
    'Employee': ['Alice', 'Bob', 'Charlie', 'David', 'Eve', 'Frank'],
    'Salary': [50000, 80000, 55000, 85000, 60000, 62000],
    'Bonus': [5000, 10000, 4000, 12000, 3000, 3500]
}
df = pd.DataFrame(data)

# Group by Department and calculate mean salary
dept_stats = df.groupby('Department')['Salary'].mean()
print(dept_stats)


2. Multiple Aggregations

You can calculate multiple statistics at once using the .agg() method.

# Calculate both mean and max salary for each department
stats = df.groupby('Department')['Salary'].agg(['mean', 'max', 'count'])
print(stats)


3. Pivot Tables

A Pivot Table is a multidimensional version of GroupBy. It allows you to group data by two or more keys, creating a grid-like summary similar to what you find in Excel.

Syntax: df.pivot_table(values, index, columns, aggfunc)

# Creating a pivot table to see Average Salary by Department
pivot = df.pivot_table(values='Salary', index='Department', aggfunc='mean')
print(pivot)


4. Multi-Level Indexing (Grouping by Multiple Columns)

You can group by more than one column to see more granular data.

# Assuming we had a 'Location' column
# df.groupby(['Department', 'Location'])['Salary'].mean()


5. Sorting Grouped Data

Often, you want to see the "top" groups (e.g., the highest-earning department).

# Sort departments by their average salary
sorted_depts = df.groupby('Department')['Salary'].mean().sort_values(ascending=False)
print(sorted_depts)
